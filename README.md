##what is myScroll?
it is a easy window scroll plug-in of javascript;

##you can use it easily!

* go everyWhere

## how to use
```javascript
var myScroll=new myScroll();
 myScroll.scroll({
    end:2000,
    time:200,
    distance: 0
  },function(ret){
    console.log(ret.top)
  })
```
